Additional Slides

Vibe Quoting UI
Vibe Quoting (Beta in 90 days for SKO)
Not a CPQ
Open access for any users and distribution
Interaction in UI is primarily via prompting
Allows file upload (e.g. XLS) to get started on a quote
Like Excel with no compliance validations
Does not require a catalog (or driven by entire world product catalog via Google partnership)

Quote Pad UI
NextGen Quote Pad (for Sales Reps)
Goal: Replace current spreadsheets sales reps are using to develop quotes
Allows basic quote interactions (CRUD, config bundles, goals seek, etc)
Generates a pre-quote
Works with Zuora CPQ only
Interaction in UI is via prompting, buttons and direct user inputs 
Allows upload of quotes from file 
Validates against established catalog, compliance rules (as in CPQ-X)

Quote Studio UI
NextGen Quote Studio (for Deal Desk)
Goal: Provide valuable information about account and the quote to a Deal Desk person to make financially sound decision
Allows to simulate quote alternatives
Provide workflow override functions
Enable interactions between sales and deal desk on a quote
Provide backend information about account
Provide profitability information such as margin and estimated break-even-point
Interaction in UI is via prompting, buttons and direct user inputs 
Validates against established catalog, compliance rules (as in CPQ-X)
